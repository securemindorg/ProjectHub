# ProjectHub
The start of a basic application for managing notes and todo lists for projects. 

# Install
1.) npm install
2.) npm run dev

# Features
- [x] MVP
- [x] Add Tasks / Delete Task / Mark Task As Complete
- [x] Add Note / Delete Note
- [x] Add Project
- [ ] Delete Project
- [x] Rename Project
- [x] Edit Note
- [x] Edit Task
- [ ] Markdown Editor
- [x] Login
- [ ] Settings Page
- [x] Admin Page
- [x] Dark Mode
- [x] Registration Page
- [x] User management
- [ ] Mobile friendly
