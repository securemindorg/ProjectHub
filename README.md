# ProjectHub
The start of a basic application for managing notes and todo lists for projects. 

# Install
1.) npm install
2.) npm run dev

# Features
- [x] MVP
- [x] Add Tasks / Delete Task / Mark Task As Complete
- [x] Add Note / Delete Note
- [x] Add Project
- [ ] Delete Project
- [ ] Rename Project
- [ ] Edit Note
- [ ] Edit Task
- [ ] Markdown Editor
- [ ] Login
- [ ] Settings Page
- [ ] Dark Mode
- [ ] Registration Page
- [ ] User management
- [ ] Mobile friendly
